Right shift (>>) operator is equivalent to division by 2
Want toÂ divide a numberÂ by 2 quickly. Here you go, use the bitwise right shift operator to divide an integer by 2. Each right shift operation reduces the number (operand) to its half.
Example:

Left shift (<<) operator is equivalent to multiplication by 2
Similar to division, you can use a bitwise left shift operator to quickly multiply a number by the power of 2 and then taking the floor of the value. Each left shift makes doubles the number (operand).
Example:
#include <stdio.h>int main() { int a = 12; // Use bitwise left shift to multiply // number by power of 2 printf("12 * (2^1) => %d\n", (a << 1)); printf("12 * (2^2) => %d\n", (a << 2)); printf("12 * (2^3) => %d\n", (a << 3)); return 0; }

Output:
12 * (2^1) => 24 12 * (2^2) => 48 12 * (2^3) => 96

Count set bits
// Naive Solution // Time: O(total number of bits) void countbit(int n){ int res = 0; while(n>0){ if(n%2 == 0) res++; n = n/2; } } // Brian and Kerningham Algorithm // Time: O(total number of set bits) void countbit(int n){ int count = 0; while (n) { n = n & (n - 1); count++; } return count; } // Method 3 (Lookup Table) // Time: O(1) int table[256]; // recursive function to count set bits void initialize() { // To initially generate the table algorithmically: table[0] = 0; for (int i = 0; i < 256; i++) { table[i] = (i & 1) + table[i / 2]; } } int countSetBits(int n) { return table[n & 0xff] + table[(n >> 8) & 0xff] + table[(n >> 16) & 0xff] + table[n >> 24]; }

Tip: The power of two numbers has one set bit in binary.
// Naive Solution bool isPow2(int n){ if(n == 0) return false; while(n !=1 ){ if(n % 2 != 0) return false; n /= 2; } return true; } // Efficient Solution bool isPow2(int n){ if(n == 0) return false; return (n&(n-1)==0); } // One line efficient solution bool isPow2(int n){ return (n!=0) &&(n&(n-1)==0); }
